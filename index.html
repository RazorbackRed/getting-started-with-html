Major Parts of the Web

The web mis made up of a number of different pieces. I am part of the web, so is my computers. Computer browsers are connnected to servers using HTTP. The final part of the web are the servers, which are very much like the hardware we use at home, but are optimized for hosting files.

HTML- Hypertext Markup Language is a type of document on the web. HTML is made up of these elements;

1. Text of content - This is what you see.

2. Markup - This is what it looks like.

3. References to other documents - This is usually made up of images and video.

4. Links to other pages - These take you to other pages on the web.

HTML Markup

HTML markup langage is made up of an opening tag, content and usually a closing tag. The closing tag includes a "/" to indicate the close of the tag. This entire sentence is called an element. Another important part of markup are tags. Tags provide direction regarding how the text should look (e.g., bold, italics, underlined).

Stupid Computers

Computers do not think on their own, they require a set of instructions for how to act. It is important to ensure that the instructions are precise and correct because the computer will follow whatever it is told to do, regardless of how incorrect those instructions are. Care with how we program is essential to avoid these mistakes.

Inline and Block

Examples of Inline elements are line breaks, bold, italics. Block elements have an invisible box around them that has height and width.
